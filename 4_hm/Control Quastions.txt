2)C++ предлагает способ сочетания преимуществ функций со скоростью кода, написанного на месте: встраиваемые (inline) функции. 
Ключевое слово inline используется для запроса у компилятора обработки вашей функции как встраиваемой функции. 
Когда компилятор компилирует ваш код, все встраиваемые (inline) функции разворачиваются на месте, 
то есть вызов функции заменяется копией содержимого самой функции, что устраняет накладные расходы на вызов функции 

3)Аргумент по умолчанию – это такой аргумент функции, который программист может не указывать при вызове функции. 
Аргумент по умолчанию добавляется компилятором автоматически вместо пустой принимаемой переменной, т.е. в теле фукнции при передаче параметров
может возникнуть ошибка, если фукнция не получила нужно количество аргументов. Чтобы избежать данной ситуации,
можно воспользоваться аргумнтов по умолчанию.
