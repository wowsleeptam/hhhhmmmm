1) Вариативные шаблоны используются в тех моментах, когда
написание фукнции должно быть обобщено на произвольное количество
принимаемых данных, у которых могут быть разные типы

2) Чтобы обработать пакет из аргументов, можно воспользоваться
методом, который заключается в том, чтоб разделить пакет из n параметров в фукнции на
один элемент, стоящий в начале, и пакет из n-1 элементов. Дальше действуем по рекурсии, пока не
дойдем до базы рекурсии (когда пакет из фукнций состояит из 0 элементов)

3)Количество параметров в пакете можно вычилсить, используя оператор sizeof...

4) Чтобы не заморачиваться с рекусивным раскрытием пачки в с++ ввели свертку.
Она упрощает код, т.к. проводит операции унарных и бинарных операторов с пачкой данных буквально в одну строчку.
Как говорилось выше, есть унарные операторы  (values + ...); и бинарные операторы existingMask |= ... |= flags.

5) Динамический полиморфизм - при компиляции программе очевидно с каким типом она будет работать в данной ситуации
Статический полиморфизм - на этапе компиляции программа сама подбирает специфичный код для каждого используемого типа в программе
